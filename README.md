# Fast Library System (C++)

This project is a simple Library Management System implemented in C++ using Structures and Dynamic Memory Allocation.

>> Features
- Create shelves
- Add new shelves dynamically
- Add books to shelves
- Remove books by ID
- Display all shelves and books
- Find the most expensive book
- Merge two shelves
- Remove empty shelves

 >>Concepts Used
- Structures
- Dynamic Memory Allocation
- Pointers
- Arrays
- Menu-driven programming

## How to Run

Compile:
g++ main.cpp -o library

Run:
./library
